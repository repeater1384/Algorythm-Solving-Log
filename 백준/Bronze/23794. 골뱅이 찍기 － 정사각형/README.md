### 문제 설명

<p>첫째 줄과 $N+2$번째 줄에는 골뱅이 $N+2$개를 출력한다.</p>

<p>둘째 줄부터 $N+1$번째 줄까지는 예제 출력과 같은 방식으로 골뱅이 $2$개와 공백 $N$개를 출력한다.</p>

### 입력 

 <p>첫째 줄에 정수 $N$($1 \leq N \leq 100$)이 주어진다.</p>

### 출력 

 <p>첫째 줄부터 $N+2$번째 줄까지 차례대로 골뱅이를 출력한다.</p>
